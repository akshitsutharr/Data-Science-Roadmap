## ðŸ“ Phase 1: Linear Algebra
**Duration:** 4-6 weeks | **Importance:** â­â­â­â­â­ (Critical)

### Why Linear Algebra?
Linear algebra forms the backbone of machine learning. It enables data representation as vectors and matrices, dimensionality reduction (PCA), neural network operations, and most ML algorithms.

### Topics to Cover

#### 1.1 Fundamentals
- Scalars, Vectors, and Matrices
- Vector Operations (Addition, Subtraction, Scalar Multiplication)
- Matrix Operations (Addition, Multiplication, Transpose)
- Vector Norms (L1, L2, Euclidean Distance)
- Dot Product and Cross Product

#### 1.2 Intermediate Concepts
- Linear Combinations and Span
- Linear Independence and Dependence
- Basis Vectors and Change of Basis
- Linear Transformations
- Matrix Rank and Null Space
- Matrix Inverse and Determinants

#### 1.3 Advanced Concepts
- Eigenvalues and Eigenvectors
- Singular Value Decomposition (SVD)
- Principal Component Analysis (PCA)
- Orthogonal and Symmetric Matrices
- LU and QR Decomposition

### Why Calculus?
Calculus is essential for understanding how machine learning models learn through optimization. Gradient descent, backpropagation, and model training all rely on calculus concepts.

### Topics to Cover

#### 2.1 Single Variable Calculus
- Limits and Continuity
- Derivatives and Differentiation Rules
- Chain Rule
- Applications of Derivatives
- Integration Basics

#### 2.2 Multivariate Calculus
- Partial Derivatives
- Gradient and Directional Derivatives
- Hessian Matrix
- Multiple Integrals
- Jacobian Matrix

#### 2.3 Applied Calculus for ML
- Cost/Loss Function Analysis
- Gradient Descent Algorithm
- Convex and Non-Convex Functions
- Taylor Series Approximation
- Optimization with Constraints (Lagrange Multipliers)

### Why Probability?
Probability theory is fundamental to machine learning for understanding uncertainty, building probabilistic models, Bayesian inference, and evaluating model performance.

### Topics to Cover

#### 3.1 Basic Probability
- Sample Space and Events
- Probability Rules (Addition, Multiplication)
- Conditional Probability
- Independence
- Bayes' Theorem

#### 3.2 Random Variables
- Discrete vs Continuous Random Variables
- Probability Mass Function (PMF)
- Probability Density Function (PDF)
- Cumulative Distribution Function (CDF)
- Expected Value and Variance

#### 3.3 Probability Distributions
- Bernoulli and Binomial Distribution
- Poisson Distribution
- Normal (Gaussian) Distribution
- Uniform Distribution
- Exponential Distribution
- Multivariate Distributions

#### 3.4 Advanced Topics
- Joint, Marginal, and Conditional Distributions
- Central Limit Theorem
- Law of Large Numbers
- Maximum Likelihood Estimation (MLE)
- Maximum A Posteriori (MAP) Estimation

## ðŸ“Š Phase 4: Statistics
**Duration:** 4-5 weeks | **Importance:** â­â­â­â­â­ (Critical)

### Why Statistics?
Statistics enables you to analyze data, draw conclusions, test hypotheses, and validate machine learning models.

### Topics to Cover

#### 4.1 Descriptive Statistics
- Measures of Central Tendency (Mean, Median, Mode)
- Measures of Dispersion (Range, Variance, Standard Deviation)
- Percentiles and Quartiles
- Skewness and Kurtosis
- Data Visualization (Histograms, Box Plots)

#### 4.2 Inferential Statistics
- Sampling Techniques
- Sampling Distributions
- Confidence Intervals
- Point Estimation
- Standard Error

#### 4.3 Hypothesis Testing
- Null and Alternative Hypotheses
- Type I and Type II Errors
- p-values and Significance Levels
- Z-tests and T-tests
- Chi-Square Tests
- ANOVA

#### 4.4 Regression Analysis
- Simple Linear Regression
- Multiple Linear Regression
- Correlation and Covariance
- R-squared and Adjusted R-squared
- Residual Analysis

#### 4.5 Bayesian Statistics
- Prior and Posterior Distributions
- Bayesian Inference
- Naive Bayes Classification
- Bayesian vs Frequentist Approaches
